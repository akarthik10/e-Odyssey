e-Odyssey is a website completely designed in Google Apps Scripts.
The website has Google Spreadsheet backend, which means users can update data on their Spreadsheet (MS Office Excel or other compatible software) and instantly see updated data reflected in the main website.
The spreadsheet back-end adds flexiblilty, as one with no programming background can update the whole website.
Simple markup language preferred is in updating spreadsheets.
The spreadsheet contains data of MainPage, UpcomingEvents, UserDatabase, and Remote Blog fetched posts.

See http://e-odyssey.appspot.com
